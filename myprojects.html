Band Name Generator

I created a band name Generator by using python .This is a very simple program and also this is my first project.In this program user inputs city name and surname then program slice the city name and surname and merge the sliced parts and print it as a band name.

The python code of this project is shown below:


 print("Welcome to band name generator")
 City_name=input("Enter your city name\n")
 surname=input("Enter your Surname\n")
 band=City_name[0:5]
 Band_n=surname[0:5]
 Band_name=band+Band_n

 print("band name for you is", Band_name ,'band')

Tip Calculator

I created a tip money Calculator using python,which calculates tip money from the bill money contributers.

The python code of this project is shown below: 

            
print("Welcome to the tip Calculator")
bill = float(input("What was your total bill ?\n"))
split = float(input("How many people to split the bill\n"))
tip_percent = float(input("What percentage tip would you like to give?\n 5,10 or 15\n"))
percent=bill*(tip_percent/100)
new_bill=percent+bill
bill_split=new_bill/split
print("Each person should pay:",int(bill_split))

BMI Calculator

I created a BMI calculator using python,in this program users can check their BMI by inputing their Height im metres and weight in kilograms

The python code of this program is shown below:

            
print("Welcome in the BMI Calculator")
weight=float(input("Enter your Weight\n"))
height=float(input("Enter your Height\n"))
BMi=weight/(height*height)
 

if BMi<18.5:
    print("Your bmi is ",round(BMi),"You are underweight")
elif BMi>18.5 and BMi<25:
        print("Your bmi is",round(BMi,"You have normal weight"))
elif BMi>25 and BMi<30:
        print("Your bmi is ",round(BMi),"Your are slightly overweight")
elif BMi>30 and BMi<35:
        print("Your bmi is ",round(BMi),"You are obese")
elif BMi>35:
        print("Your bmi is ",round(BMi),"You are clinically obese")
